Setup
-----

Copy `config.js.dist` to `config.js`

Change the value of `secret` and run it with `node app.js`.

Dependencies
------------

You can `npm install` all of these:

  - `backbone`
  - `connect`
  - `connect-redis`
  - `express`
  - `expresso`
  - `jade`
  - `mailer`
  - `mongodb`
  - `mongoose`
  - `redis`
  - `underscore`
  - `redis-completer`
